题目地址：https://leetcode-cn.com/problems/jump-game-ii/submissions/

```java
class Solution {
    public int jump(int[] nums) {
        int length = nums.length;
        int end = 0;
        int maxPosition = 0; 
        int steps = 0;
        for (int i = 0; i < length - 1; i++) {
            maxPosition = Math.max(maxPosition, i + nums[i]); 
            if (i == end) {
                end = maxPosition;
                steps++;
            }
        }
        return steps;
    }
}
```

**复杂度分析**

- 时间复杂度：O(n)，其中 n 是数组长度。
- 空间复杂度：O(1)。
